Flutter 调试模式白屏而 flutter run 正常的解决方案

• 2020-09-26

这个问题困扰了我好久好久……少说有 2 个月，之前直接放弃选择去做其他的东西了。具体的问题表现为：

  • flutter run 可以正常在真机、模拟器上运行
  • 使用 Android Studio 或是 Visual Studio Code 都会卡在 installing build/app/outputs/flutter-apk/app-debug.apk... 且不出现报错，此时机器上的 apk 白屏

issue：https://github.com/flutter/flutter/issues/22338

尝试了 开发者选项 -> 调大 log buffer size、flutter clean、硬件加速等等无果之后，最后发现是 代理 的问题！！！

在 .bashrc 或是其他你用的 shell 里给 NO_PROXY 加上 localhost,127.0.0.1 就可以了：

export NO_PROXY=localhost,127.0.0.1

这调试的体验真的是太一波三折了，直接劝退
